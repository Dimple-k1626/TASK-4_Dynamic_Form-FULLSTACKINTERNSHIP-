🚀 Advanced Dynamic Registration Form
📌 Project Overview

This project is an Interactive Registration Form built using HTML, CSS, and JavaScript.
It demonstrates advanced DOM manipulation, real-time validation, and modern UI design inspired by Apple and Stripe signup pages.

The application focuses on creating an interactive user experience using dynamic form fields and real-time feedback.

✨ Features
🔐 Live Password Strength Meter

Displays password strength dynamically based on:

Length

Uppercase letters

Numbers

Special characters

📊 Form Completion Progress Bar

A progress bar updates automatically as the user fills the form.

📱 Dynamic Phone Number Fields

Users can dynamically add or remove phone number fields.

⚡ Client-Side Routing

The application switches between:

Registration Page

Success Page

without refreshing the browser.

🎨 Modern UI Design

The UI includes:

Glassmorphism card design

Floating input labels

Smooth animations

Gradient buttons

Responsive layout

🛠 Technologies Used

HTML5

CSS3

JavaScript (DOM Manipulation)
